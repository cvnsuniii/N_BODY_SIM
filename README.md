#N body simulator-
a gravitational simulator using newton's equations of gravity and motion. These Sims are used to analyze the influence between N bodies which helped discover neptune and is the foundation for planet 9. This version of software provides capabilities for simulating upto 99 bodies at once 1 device can have 40 Sims stored in local storage. This version gives only animation at the end which is a projection method with sim parameters which user can set. Future versions with more 3d effects, complex object controls, collision mechanics and animation features may be developed soon. Also the app will be deployed by aac Web team soon on aac.nitk.ac.in
